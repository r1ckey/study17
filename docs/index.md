## このサイトについて

データエンジニア（DE）としてのキャリアを築くための学習記録です。

**目指す姿**

```

```

## 学習の進め方

このサイトは **いつでもどこでもスマホで見れる** ことを前提に作っています。

- 通勤中 → 概念ページをさっと読む
