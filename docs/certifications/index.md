# 資格ロードマップ

## 取得順序と目的

```
フェーズ1   GCP ACE          ← 今ここ（70%完了）
  ↓
フェーズ2   Databricks Associate
  ↓
フェーズ3   Azure DE Associate
  ↓
将来        統計検定準1級
```

---

## 各資格の概要

| 資格 | 有効期限 | 難易度 | 目的 |
|------|---------|-------|------|
| GCP ACE | 2年 | ★★☆ | 完走実績・会社要件クリア |
| Databricks Associate | 2年 | ★★☆ | DE主武器の証明 |
| Azure DE Associate | 1年（更新容易）| ★★★ | 案件直結・市場価値 |
| 統計検定準1級 | 5年 | ★★★★ | 転職時の差別化 |

---

## 会社要件との対応

| 会社の要求 | 対応 | 理由 |
|-----------|------|------|
| クラウド資格 | GCP ACE + Azure DE | 2クラウド対応済み |
| OSS DB Silver | 取らない | DEでなくインフラへの配属リスク |
| セキュリティ資格 | Azure DEの中でカバー | 追加取得不要 |
| 得意言語1つ | Python（PySpark）| 全案件タイプで通用 |
